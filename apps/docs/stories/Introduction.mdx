# 🍫 Brownie UI

**A beautiful, modern UI component library for React**

Brownie UI is a collection of beautifully designed components that you can copy and paste into your apps. Inspired by Pinterest designs and built with shadcn/ui compatibility in mind.

## ✨ Features

- 🎨 **Beautiful by default** - Carefully crafted components with great aesthetics
- 🔧 **Fully customizable** - Built with Tailwind CSS and CVA for easy customization
- ♿ **Accessible** - WCAG compliant with proper ARIA labels
- 📱 **Responsive** - Mobile-first design approach
- ⚡ **Performance** - Tree-shakeable and lightweight
- 🔥 **TypeScript** - Full type safety

## 🚀 Getting Started

### Installation

```bash
# Via shadcn CLI (coming soon)
npx shadcn add https://brownie-ui.dev/r/button

# Or via npm
npm install @brownie-ui/button @brownie-ui/core
```

### Usage

```tsx
import { Button } from '@brownie-ui/button';

function App() {
  return (
    <Button variant="primary" size="lg">
      Get Started
    </Button>
  );
}
```

## 🎨 Design System

Our design system is inspired by rich chocolate tones and modern aesthetics:

- **Primary Colors**: Warm browns (`#a18072` to `#2d1f1a`)
- **Neutrals**: Stone grays for backgrounds and text
- **Accents**: Amber and orange for highlights
- **Glassmorphism**: Translucent overlays for modern UI patterns
